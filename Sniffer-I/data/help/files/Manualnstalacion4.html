Manual de instalacion

Para el desarrollo de la aplicación, se ha  utilizado la JDK 6.0 (Java Development Kit). Para la ejecución sin embargo, basta con instalar el entorno de ejecución, esto es el JRE 6.0 (Java Runtime Environment).

Requisitos hardware y software para Java:


  • Requisitos Software (en cuanto a los sistemas operativos):
    • Windows 95, Windows NT 4.0 y versiones superiores.
    • Sun Solares 2.4, 2.5, 2.5.1, 2.6 sobre SPARC y Sun Solares 2.5, 2.5.1, 2.6 sobre x86 y versiones superiores.
  • Requisitos Hardware (en cuanto al espacio en disco)
    • Una copia normal del instalador precisa de 120 MB de espacio en disco.
