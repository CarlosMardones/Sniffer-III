Ejecucion en Modo Comando

La opcion en modo comando presenta una pequeña ayuda donde  visualizar las principales funciones ejecutables.

Para poder sacar el máximo partido a este modo de ejecución, será necesario generar los ficheros de parametrización desde el entorno gráfico. Una vez generados (a elección del usuario) se podrán modificar con un editor de texto. El formato de los mismos, viene especificado en la documentación del programa.

Opciones más frecuentes del modo comando en forma de scripts.

  • Realización de una captura (Windows)

set ruta=C:\Sniffer-I

set ejecutable=Sniffer.jar

set param=-command -scan "C:\Sniffer-I\files\parametrizacion\P_Captura.xml"

set mvm=0

cd %ruta%

set PATH=.;%PATH%

set java=java

if not '%mvm%' == '0' set java=java -Xmx%mvm%m

%java% -jar %ejecutable% %param%


  • Realización de una captura (Linux)

ruta="C:\Sniffer-I"

ejecutable=sniffer.jar

param="-command -scan C:\Sniffer-I\files\parametrizacion\P_captura.xml"

mvm=0

cd $ruta

#set PATH=.;%PATH%

if [[ $mvm = "0" ]]

then

java=java

else

java="java -Xmx"$mvm"m"

fi

$java -jar $ejecutable $param
