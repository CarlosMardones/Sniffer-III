Generar Script

Una vez que  realizadas las configuraciones y parametrizaciones comentadas en los apartados anteriores es el momento, de generar los scripts que van a permitir ejecutar la aplicación en modo comando para así obtener el máximo rendimiento de esta herramienta.

Esta ventana está subdividida en cuatro apartados:

  • Datos de entrada, nos  seleccionar tanto el fichero de preferencias de captura como el nombre del script que se desee generar.

  • Tipo de script que se desea generar, puede ser de tres tipos, captura, exportación o exportación desde fichero.

  • Cantidad de Memoria que se desea asignar a la Máquina Virtual Java.

  • Y por último el Sistema operativo en el que se ejecutara la aplicación en modo comando
